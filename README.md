# 🧩 Sudoku Solver

A modern web-based Sudoku game built using "Constraint Satisfaction Problem (CSP)" techniques with a clean UI.


## 🚀 Features
- 🧠 Sudoku solver using CSP (Backtracking)
- 🧩 Auto puzzle generator
- 🎮 Interactive 9x9 grid
- ✨ Popup notifications (modern UX)
- 🎨 Stylish UI with JetBrains Mono font


## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
